# 🏡 StayNest – Airbnb-Inspired Booking Platform

StayNest is a full-stack property booking web application inspired by Airbnb. It enables users to browse property listings, register and log in securely, upload property images, manage listings, and book stays. The application is built using Node.js, Express.js, MongoDB, and EJS, following the MVC architecture.

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/adarsh73808/staynest-booking.git
cd staynest-booking
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env` file in the root directory:

```env
DB_PATH=your_mongodb_connection_string
PORT=3003
```

### 4. Start the Application

```bash
npm start
```

### 5. Open in Browser

```bash
http://localhost:3003
```
