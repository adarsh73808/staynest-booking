# StayNest 🏡 – Airbnb-Inspired Booking App

**StayNest** is a full-stack web application built with **Node.js**, **Express**, **MongoDB**, and **Tailwind CSS**. It allows users to register, list properties, search for rentals, and book stays. This project is inspired by Airbnb and demonstrates strong backend logic with a modern UI using Tailwind CSS.

---

## 🚀 Features

- 🔐 User Authentication (Register/Login)
- 🏠 Add, Edit, Delete Property Listings
- 📍 Search by Location
- 📆 Booking with Dates
- 🖼️ Upload Property Images
- 💾 MongoDB for Database Storage
- 📱 Fully Responsive with Tailwind CSS
- 🧭 Express Routing & Controllers

---

## 🛠️ Technologies Used

- **Frontend**: HTML, Tailwind CSS, JavaScript, EJS Templates
- **Backend**: Node.js, Express.js
- **Database**: MongoDB + Mongoose
- **Authentication**: bcrypt + express-session / JWT (as per your setup)
- **Image Uploads**: Multer
- **Tools**: Git, GitHub
